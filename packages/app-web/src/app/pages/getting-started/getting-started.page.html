Getting Started

Powerful RSS feed middleware

Free Open-Source Service to create, filter and manage your feeds.

Open Source

Checkout github to fork, create an issue or feature request and self-hosting.

Features

  • Content enrichment (e.g. Fulltext, Media Detection)
  • Web-to-Feed with JavaScript Support
  • Aggregation of multiple feeds into one
  • Plugins
  • Privacy first (e.g. inline images)
  • Filtering
  • Feed Format Conversion
  • CLI
